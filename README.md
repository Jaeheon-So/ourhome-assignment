# ourhome-assignment

[배포링크](https://sjh-ourhome-assignment.netlify.app/)

따로 실행을 하려면

1. Github clone 시
   ```bash
   git clone https://github.com/Jaeheon-So/ourhome-assignment.git
   cd ourhome-assignment
   npm install
   npm run dev
   ```
2. 파일 다운로드 시 해당 디렉토리로 이동 후
   ```bash
   npm install
   npm run dev
   ```
